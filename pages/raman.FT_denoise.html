raman.FT_denoise¶

raman.FT_denoise(y, cutoff=1, filter='rect')¶

Apply a Fourier low pass filter to the data to remove high frequency noise.

Parameters:
  • y (array) – Input spectrum. If y is a list or 2d-array of spectra, filter every spectrum.
  • cutoff (int) – Low pass cutoff position taken from the high frequency side in array indices (0 means no filtering, default = 1).
  • filter (str) –

    Type of step function to apply as filter. Currently supported are:

    • ‘cos2’ - A cosine squared.
    • ‘linear’ - A linear onset.
    • ‘rect’ - A rectangular step function (default).
Returns:

Filtered spectrum with same shape as y.
