profilometer.convert¶

profilometer.convert(filename)¶

Convert files from sdf to xyz format. Saves the converted file using the same filename and ”.xyz” as file ending.

Parameters:filename (str) – Filename to convert. Uses glob to support wildcards and regular expressions.
