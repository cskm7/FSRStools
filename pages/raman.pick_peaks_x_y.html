raman.pick_peaks_x_y¶

raman.pick_peaks_x_y(x, y=None)¶

Utility function to quickly extract x and y coordinates of points from data by double clicking in a plot window. For each double click, the x and y coordinates are printed on stdout. After the function is executed, the script terminates.

Use this function to get interpolation points as input for interpolated_bl_correction(), for example.

Parameters:
  • x (array) – x-axis or data if y is None.
  • y (array) – Data (optional). If None, the function displays the data vs indices.
