raman.at¶

raman.at(x, y, x0)¶

Return the value of y with x-coordinate closest to x0.

Parameters:
  • x (array) – x-axis.
  • y (array) – Data (same shape as x).
  • x0 (float) – x-coordinate of desired data point. If x0 is a list or array, return an array of y values.
Returns:

Data point with x-coordinate closest to x0.
