raman.interpolate¶

raman.interpolate(x, y, N=0, kind='linear')¶

Convenience wrapper around SciPy’s interp1d() function.

Parameters:
  • x (array) – x-axis with non-equidistant sampling points
  • y (array) – data sampled on x (same shape as x)
  • N (int) – Number of additional points that should be added per interval (default = 0).
  • kind (str) – Type of interpolation to be used (default = ‘linear’). This parameter is directly passed to interp1d().
Returns:

(x’, y’) x- and y-arrays sampled equidistantly.
