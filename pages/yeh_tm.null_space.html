yeh_tm.null_space¶

yeh_tm.null_space(A, eps=0.0001)¶

Compute the null space of matrix A which is the solution set x to the homogeneous equation Ax = 0.

Parameters:
  • A (matrix) – Matrix.
  • eps (float) – Maximum size of selected singular value relative to maximum.
Returns:

Null space (list of vectors) and associated singular components.
